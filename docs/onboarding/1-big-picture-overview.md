# Big Picture Overview
{: .no_toc }

**Who this is for:** First-time clinicians using the platform for the first time.

---

## The Problem We Solve

When you receive lab results for a patient, the numbers come with reference ranges. These ranges tell you whether a value is "normal" or "abnormal."

But here's the challenge: those reference ranges were derived from broad population studies. They tell you when someone is clearly sick. They don't tell you when someone is thriving.

A TSH of 3.8 might be "normal" by conventional standards. But for a 32-year-old woman trying to conceive, it might be suboptimal. The conventional range doesn't capture that nuance.

This platform exists to bridge that gap.

---

## What Functional Interpretation Means

Functional interpretation is not a rejection of conventional medicine. It is an additional lens.

Conventional ranges answer: "Is this patient diseased?"

Functional ranges answer: "Is this patient optimized?"

Both questions are valid. Both deserve answers. This platform provides both.

When you view a patient's results, you see the conventional classification alongside the functional classification. Nothing is hidden. Nothing is replaced. You get more information, not different information.

---

## How the System Applies Ranges

The platform uses something called a Named Range Set.

Think of a Named Range Set as a complete reference book. It contains definitions for hundreds of lab markers, each with boundaries for what is considered low, optimal, or high.

Your clinic selects which reference book to use. This is a foundational decision that shapes how all results are evaluated.

Some clinics focus on general wellness. Others specialize in athletic performance. Others work with fertility patients. Each of these contexts may benefit from a different reference book — a different Named Range Set.

Once your clinic has selected its Named Range Set, the platform uses that framework consistently for all patients. Within that framework, the system finds the most appropriate range for each patient based on their demographics: age, sex, pregnancy status, and so on.

This is not guesswork. It is systematic, reproducible, and auditable.

---

## What the System Does Not Do

The platform does not diagnose. It does not prescribe. It does not override your clinical judgment.

It classifies lab values according to a transparent framework. It explains why each classification was made. It surfaces patterns you might want to investigate.

You remain the clinician. The platform is a tool that organizes information and applies reference frameworks consistently.

---

## Transparency and Auditability

Every classification the platform makes is traceable.

You can see exactly which Named Range Set was used. You can see which specific range definition was applied. You can see why that definition was selected over alternatives.

If you ever need to explain to a patient, a colleague, or a reviewer why a particular value was flagged, the platform provides that explanation. Nothing is a black box.

## Key Takeaways

- Conventional lab ranges tell you about disease. Functional ranges tell you about optimization.
- The platform provides both views, clearly labeled.
- Your clinic selects a Named Range Set — a complete reference framework.
- The system applies that framework consistently and explains every decision.
