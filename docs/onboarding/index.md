## Core Concept

Throughout this series, one concept remains central:

> **Named Range Sets allow entire interpretive frameworks to be swapped.**
>
> Your clinic selects a worldview (the Named Range Set). Patient demographics determine which specific range applies within that worldview. Interpretation is layered on top — without changing numeric ranges.

These three layers are distinct:

1. **Layer 1:** Named Range Set selection (clinic worldview)
2. **Layer 2:** Context resolution within that set (patient demographics)
3. **Layer 3:** Interpretation overlays (symptoms, patterns, AI)

Symptoms influence interpretation. They do not influence numeric ranges.
